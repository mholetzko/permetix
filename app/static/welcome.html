Welcome to Permetrix

Permetrix is a cloud-native platform for application licensing. Vendors issue licenses once, customers consume them everywhere—backed by real-time observability, budget guardrails, and secure client SDKs.

For Vendors

Provision products to customers, set commit/overage limits and prices, export usage, and monitor health in real time. White‑label and custom domains supported.

For Customers

Integrate via API keys and SDKs, protect budgets with max spend, and get transparent usage and costs. No infrastructure to run.

Security

API keys + vendor secrets + HMAC signatures by default. Roadmap includes mTLS and hardware-backed keys.

Observability

Built-in metrics, logs, and real-time SSE dashboards. Long-term analytics via Prometheus and Grafana Cloud.
